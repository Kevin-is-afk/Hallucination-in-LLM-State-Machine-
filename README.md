# Hallucination-in-LLM-State-Machine-
Hybrid hallucination detection system using RoBERTa-MNLI combines model predictions with eigenvector-based uncertainty scoring. An optimized threshold is applied to generate final predictions, evaluated using metrics like Accuracy, F1-score, and ROC-AUC.

## Description
This project detects hallucinations in QA pairs using a hybrid scoring method combining:
- RoBERTa-MNLI probabilities
- Eigenvector-based uncertainty scoring

## Features
- Hybrid scoring
- Automatic threshold optimization
- ROC curve evaluation
